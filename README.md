# 天翼云桌面保活工具 (Rust 版本)

这是一个用 Rust 编写的天翼云桌面保活工具，用于保持云桌面连接活跃，防止自动断开。

## 功能特性

- 自动登录天翼云桌面
- 支持多个云桌面同时保活
- WebSocket 连接保持活跃
- 验证码自动识别
- 配置文件存储（密码加密）

## 使用方法

### 首次运行

```bash
cargo run --release
```

首次运行时会提示输入账号和密码，配置会自动保存到 `config.json` 文件中。

### 后续运行

```bash
cargo run --release
```

程序会自动读取配置文件并登录。

## 配置文件

配置文件 `config.json` 存储在项目根目录，包含以下信息：

- `user_account`: 天翼云账号
- `password`: 密码（Base64 编码存储）
- `device_code`: 设备代码（自动生成）

## 编译

```bash
cargo build --release
```

编译后的可执行文件位于 `target/release/ctyun_keepalive`

## 依赖

- Rust 1.93.1 或更高版本
- 互联网连接

## 注意事项

- 密码以 Base64 编码存储，仅提供简单的混淆保护
- 请妥善保管配置文件
- 建议在安全的环境中使用
